# 🎬 Netflix Data Analysis Project

## 📌 Project Overview

This project analyzes Netflix’s catalog of movies and TV shows to uncover key trends in content distribution, genres, ratings, and growth over time. Using Python for data analysis and visualization, the project transforms raw data into meaningful insights.

---

## 🚀 Project Motivation

With the rapid growth of streaming platforms, understanding what type of content attracts and retains users is critical. This project was created to explore how Netflix structures its content library and to practice real-world data analysis skills relevant to business decision-making.

---

## 🎯 Business Problem

Netflix operates in a highly competitive streaming industry where content strategy plays a major role in user engagement and retention.

This project aims to answer:

* What type of content dominates the platform?
* How has Netflix expanded its content library over time?
* What genres and ratings are most common?
* What does this reveal about Netflix’s target audience?

By answering these questions, we can better understand Netflix’s content strategy and how data can support strategic decisions.

---

## 📊 Key Questions

* Movies vs TV Shows: which is more dominant?
* How has content grown over time?
* Which genres are most popular?
* What ratings are most frequent?
* What is the typical duration of movies?

## 📂 Dataset

This project uses the **Netflix Movies and TV Shows dataset from Kaggle**, which contains detailed information such as titles, genres, release years, ratings, and durations.

---

## 🧹 Data Cleaning

* Handled missing values
* Converted date columns into proper format
* Extracted year from `date_added`
* Cleaned duration column
* Processed genre categories for analysis

---

## 📈 Analysis & Visualizations

The analysis includes:

* Content type comparison (Movies vs TV Shows)
* Growth of Netflix content over time
* Genre distribution
* Ratings distribution
* Movie duration analysis
* Top countries producing content

---

## 🔍 Key Insights

* Netflix has significantly more movies than TV shows, indicating a strategy focused on high-volume content delivery.
* Content growth accelerated rapidly after 2015, aligning with global expansion.
* Drama and International Movies dominate the platform, showing a diverse and global content approach.
* Most content is rated for mature audiences, suggesting a focus on adult viewers.
* The majority of movies fall within the 80–120 minute duration range.

---

## 💡 Business Insights

* A strong focus on movies allows Netflix to frequently refresh its catalog and attract a wide audience.
* The rise of international content highlights Netflix’s global expansion strategy.
* Content ratings indicate that Netflix prioritizes adult and young adult audiences.
* Data-driven insights like these can help guide content production and acquisition decisions.
